# Calculadora-de-Consumo-de-Combust-vel
Programa que calcula o consumo médio de combustível de um veículo

## Descrição

Este projeto é uma calculadora de consumo de combustível que determina o consumo médio de um veículo com base na distância percorrida e na quantidade de combustível utilizada.  
O programa realiza o cálculo em quilômetros por litro (km/l) e exibe todos os dados da viagem, apresentando o resultado final arredondado para duas casas decimais.

**Exemplo usado no programa:**
- Distância percorrida: 300 km  
- Combustível gasto: 25 litros  

**Saída esperada (para os valores acima):**
- Consumo médio: 12.00 km/l
